# hrgm401-react-component-kit
## 1. 概要
- UIコンポーネントの汎用化を目指し、そのカタログ化を目的に作られました。
- 第一段階として汎用化の初期段階としてプロトタイプ版を作成し、現在第二段階として実用化に向けたデザインシステムへの移行・リファクタリングを行っています。
- Hard-coded CSSからTailwind v4のDesign Tokensへの置換、およびコンポーネントのAtomic Design化を進行中です。

## 2. 開発ロードマップ
### 目標
- デザインを統一する
- パッケージとして使いやすいように処理やデザイン、引数の調整
- storybookをすぐに見て分かるようにする

**Phase 1: 土台作り（Design Tokensの定義）**
- ハードコードされた値を「変数」に置き換え、システムを作る。

**Phase 2: コンポーネント実装とマイクロインタラクション**
- ボタン、入力、テーブルなどを、状態（State）を含めて修正する。

**Phase 3: 独自性（テーマ切り替え）とドキュメント化**
- 「色変え機能」の実装と、Storybook自体の見せ方の整備。

**修正箇所詳細**
| 修正ポイント | 詳細 |
| --- | --- |
| *汎用ライブラリ化* |  |
